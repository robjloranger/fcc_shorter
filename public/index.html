Shorter!

Enter a valid URL to get back a new, shorter URL.


Valid URL's are as follows:
Always either http or https
With or without www or other subdomain
And optionaly any number of subdirectories followed by a file
Examples inlcude:
http://www.google.com
or https://mail.google.com
or https://www.google.com/doodles

Use this service as an API

To use this URL shortening service in your app simply POST to this rjl.li/new with a query parameter for origin with your url-encoded domain to shorten.

Example: http://rjl.li/new/?origin=http%3A%2F%2Fwww.google.com

The sever will respond back with json in the following format:
{ "shortcode": "E4mrt5", "origin": "http://www.google.com", "shorturl": "http://rjl.li/E4mrt5" }

All requests to valid url shortcodes will redirect to the original url submitted.
